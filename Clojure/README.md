# Aprendizado de Programação Funcional com Clojure

Este repositório é dedicado aos meus estudos de **Programação Funcional** utilizando a linguagem **Clojure**. 

O objetivo principal é aprofundar os conceitos do paradigma funcional (imutabilidade, funções de alta ordem, pureza, etc.) enquanto resolvo problemas de lógica e algoritmos.

## 🚀 O Método

Para praticar a sintaxe e o raciocínio funcional, estou resolvendo desafios da plataforma **Codeforces**. 

A escolha do Codeforces se deve à variedade de problemas que exigem não apenas o conhecimento da linguagem, mas também o uso eficiente de estruturas de dados e algoritmos, o que me força a pensar "do jeito Clojure" para manipular coleções e fluxos de dados.

## 🛠️ Ferramentas Utilizadas

- **Linguagem:** [Clojure](https://clojure.org/)
- **Editor:** Sublime Text (configurado via CLI com o comando `subl`)
- **Plataforma de Desafios:** [Codeforces](https://codeforces.com/)

## 📂 Estrutura do Repositório

- `/problemas`: Contém as resoluções dos desafios, nomeadas pelo ID do problema no Codeforces (ex: `158A.clj`).
- `/conceitos`: Notas de estudo sobre conceitos fundamentais de Clojure.
